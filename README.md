Nginx + PHP-FPM base on CentOS7
===============================


## Fetures ##

- OS base on CentOS 7(https://hub.docker.com/r/centos)
- php 7.2 from remi repo and installed packages:
  - php-fpm : enabled
  - php-opcache : enabled
  - php-pdo : enabled
- nginx from official and default enabled
